# ProjectWebDev

An Angular web application for managing games and user administration.

## 🌐 Live Application

The app is hosted on **Firebase Hosting** and can be accessed here:

**[https://projectwebdev-4da93.web.app/](https://projectwebdev-4da93.web.app/)**

## About

This project is built with [Angular CLI](https://github.com/angular/angular-cli) version 20.3.2 and uses Firebase for authentication and data management.
